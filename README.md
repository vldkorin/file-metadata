# File Metadata Microservice

This is a simple file metadata microservice built with **Node.js**, **Express**, and **Multer**. It allows users to upload a file and returns metadata about the file: its name, type, and size.

## Features

- Upload a file through a form
- Get the uploaded file's:
  - Original name
  - MIME type
  - Size in bytes

## Technologies Used

- Node.js
- Express
- Multer (for handling file uploads)
- dotenv (for environment variable management)
- CORS
